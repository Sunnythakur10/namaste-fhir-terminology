About the Project

This terminology translation framework integrates NAMASTE (National AYUSH Morbidity & Standardized Terminologies Electronic) with WHO's ICD-11 (International Classification of Diseases) Traditional Medicine Module 2 within FHIR-compliant EMR systems.

It enables AYUSH practitioners to document diagnoses using their traditional terminology while automatically mapping to internationally recognized codes for research, insurance, and interoperability purposes.
